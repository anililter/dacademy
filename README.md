# d.academy – Web Sitesi

Premium bire bir eğitim spor merkezi için tek sayfa web sitesi.

## Kurulum

1. **Telefon ve WhatsApp:** `index.html` içinde `90XXXXXXXXXX` geçen tüm yerleri gerçek telefon numaranızla değiştirin (ülke kodu 90, başında 0 olmadan).
   - Örnek: `905321234567` → `tel:+905321234567` ve `https://wa.me/905321234567`

2. **Yayınlama:** `index.html` ve `styles.css` dosyalarını herhangi bir web sunucusuna veya hosting’e yükleyin. Yerel test için:
   ```bash
   # Python 3
   python3 -m http.server 8000
   # Sonra tarayıcıda http://localhost:8000
   ```

## İçerik

- **Hero:** Premium Bire Bir Eğitim, Potansiyelinizin Zirve Noktası
- **Eğitimler:** Fitness, Dövüş Sporları, Reformer Pilates, Yüzme
- **CTA:** İlk aya özel %30 indirim
- **Sabit butonlar (sağ alt):** Telefon ve WhatsApp
